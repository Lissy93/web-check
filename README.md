# web-check

เว็บโปรเจกต์สำหรับตรวจสอบสถานะเว็บไซต์และการตั้งค่าที่เกี่ยวข้อง

สรุป
- โครงการนี้ใช้สำหรับตรวจสอบสถานะเว็บไซต์ (uptime), ตรวจสอบ SSL, การตอบกลับของ API และแจ้งเตือนเมื่อมีปัญหา

เริ่มต้นใช้งา��� (แบบย่อ)
1. ติดตั้ง dependencies:
   ```bash
   npm install
   ```
2. สร้างไฟล์ตั้งค่า (ตัวอย่าง `config/example.env`) แล้วตั้งค่าตามความต้องการ
3. รันในโหมดพัฒนา:
   ```bash
   npm run dev
   ```
4. ตรวจสอบผลลัพธ์ผ่าน UI หรือ API ตามที่โครงการกำหนด

การสนับสนุนภาษาไทย
โครงการนี้รองรับข้อความแสดงผลภาษาไทยโดยใช้ไฟล์แปลภาษาที่อยู่ใน `i18n/th.json` หากต้องการปรับแก้ข้อความเพิ่มเติม ให้แก้ไฟล์ดังกล่าวแล้วเปิด PR มายังสาขา `main` หรือ `Futufe`
